Data Flow

数据流

任务从 AI 工具出发，经过 Mac/helper 归一化，最终通过 USB Serial 到达 ESP32 小屏。

STEP 01
AI 工具执行
Codex / Claude / 即梦 各自将日志写入 session/project JSONL 文件
~/.codex/sessions/*.jsonl
STEP 02
Mac/helper 轮询采集
server.mjs 每 850ms/900ms 轮询 JSONL，新增行解析后更新内存任务状态
850ms / 900ms setInterval
STEP 03
SSE 广播
状态通过 Server-Sent Events 广播给 Web 客户端和 Swift 桌宠
text/event-stream · /events
STEP 04
USB Serial → ESP32
状态打包为 JSON Lines，写入 /dev/tty.usbmodem1301
USB Serial JTAG · JSON Lines
STEP 05
ESP32 渲染
serial_task 解析 JSON Lines，触发 peekdock_screen_apply_event()
LVGL 9.x · RGB565
STEP 06
小屏动作上报
触摸横滑触发 action_event 写回 Serial，执行白名单快操或召回动画
return_to_mac · open_result
